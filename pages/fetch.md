- fetch1 /fɛtʃ/  *verb*
- 1. go for and then bring back (someone or something) for someone.
  "he ran to fetch help"
- 2. In development, **“to fetch”** generally means **to retrieve data from another source**, most commonly over a network.
- 3. tech => Core meaning **Fetch = get data from somewhere else**
	- Usually from:
		- an API (backend server)
		- a database (indirectly via API)
		- a file or external resource
