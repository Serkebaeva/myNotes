- [[preprocessor]]
  collapsed:: true
	- CSS as a language has same problems as JS, we can't organize our code into different files, and it missing a lot of useful features. And we also use a [[bundler]] and [[transpiler]] for CSS. There is a special name for this the CSS [[preprocessor]].  Its basically combination of bundler and transpiler.
	- **Preprocessor = bundler + transpiler.**
	- The most popular one of them -> [[Sass]]
- [[Frameworks]]
	- Once we have **preprocessors**, developers started to create **CSS Frameworks**.
	- One of the most popular is -> [[bootstrap]]. CSS frameworks are basically a whole bunch of css code that someone else wrote, and it helps us solve common problems. So, using a framework saves us a lot of time.
