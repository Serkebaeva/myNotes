aliases:: IP address

- Real web addresses ([[URL]]s) aren't the nice, memorable strings you type into your address bar to find your favorite websites. They are special numbers that look like this: `192.0.2.172.`
- This is called **IP address**, and it represents a unique location on the web. However, it's not very easy to remember, is it? That's why the DNS was invented. This system uses special servers that match up a web address you type into your browser to the website's real IP address. Large websites are commonly made available on multiple servers, so that they load efficiently for different users worldwide. As a result, the IP address may vary depending on where you are.
- You can use a DNS lookup tool to find IP address of a website. e.g. DNS lookup tool ->https://www.nslookup.io/
