- [[bundler]]
  collapsed:: true
	- JS as a language is missing a lot of features that other programming languages have. One of these is being able to split up our code into different files, to solve this he had to use something called a [[bundler]]. The most popular one is called [[webpack]]. Webpack lets us split up our **JS** code into many different files, and once we are ready to put it on our website, webpack will combine all of those files, or bundle all of these files into one JS file -> usually **bundle.js**.
- [[transpiler]]
	- Another tool that we use is transpiler, the most popular one is [[typescript]]. Transpiler adds extra features to JS. It lets us write an enhanced version of JS, and then once we done it will transform enhanced js back into normal, because browsers like google chrome can only understand normal JS.
